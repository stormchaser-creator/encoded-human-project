## Evidence Review

### 1. The Cholinergic Anti-Inflammatory Pathway: Core Architecture

The CAP, described across multiple entries (WS2-Defense-Cholinergic-Anti-Inflammatory-Pathway-P1; WS5-Defense-Cholinergic-Anti-Inflammatory-Pathway-D1), represents Kevin Tracey's 2000 paradigm shift: the nervous system does not merely observe inflammation but actively controls it in real time through a dedicated neural circuit. The pathway proceeds: peripheral cytokine detection by vagal afferents → brainstem processing → vagal efferent activation → celiac ganglion → splenic nerve → norepinephrine release → ChAT+ T cell acetylcholine production → alpha7 nicotinic acetylcholine receptor (alpha7nAChR) activation on macrophages → NF-kB suppression → reduced TNF-alpha, IL-1beta, IL-6 production.

The multi-node architecture is critical: failure at any point in this chain produces the same downstream outcome — unbraked inflammatory cytokine production — but via different mechanisms requiring different interventions. This is the central clinical diagnostic problem implicit in the entry title: 'unresponsiveness' to VNS could reflect pre-ganglionic failure (insufficient vagal efferent signal), ganglionic failure (splenic nerve dysfunction), cellular failure (depleted ChAT+ T cells), or receptor failure (alpha7nAChR desensitization or downregulation).
