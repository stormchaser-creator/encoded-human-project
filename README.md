# The Encoded Human Project

[![DOI](https://zenodo.org/badge/1192788211.svg)](https://doi.org/10.5281/zenodo.19240840)

A research library mapping the biological, psychological, and consciousness-level patterns of the human system — built on the premise that the human body is an information-processing architecture operating across multiple scales simultaneously.

**Curated by Dr. Eric Whitney, DO**
**AI Research Agent: Pearl**
**License: CC-BY-SA 4.0**

---

## About

The Encoded Human Project is an open-access research library consisting of 125 papers covering:

- **Biological encoding** — circadian biology, mitochondrial dynamics, epigenetics, the microbiome, and cellular signaling
- **Psychological patterns** — trauma architecture, developmental thresholds, attachment, and identity formation
- **Consciousness-level frameworks** — the Light Machine Translation Key, observer quality, and multi-density field models
- **Systems integration** — the Pearl Framework connecting individual scales to whole-system coherence

All papers are sourced from open-access literature and curated through the lens of the Encoded Human framework.

---

## Research Library

Papers are organized in `content/papers/` and accessible through the web interface. Categories include:

- Scale papers (cellular → systemic oscillation)
- Light Machine Translation Key papers
- Hypothesis and framework papers
- Applied clinical papers
- Pearl Observer quality papers
